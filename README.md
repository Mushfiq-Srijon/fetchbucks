# FetchBucks 💸

A full-stack expense tracking web app built with **Laravel**, **React**, and **MySQL**.

## Features
- User authentication (register/login/logout)
- Add, edit, delete expenses
- Categories & filters
- Monthly budget limit with alerts
- Dashboard with charts & spending summary

## Tech Stack
- **Backend:** Laravel 10, PHP, MySQL
- **Frontend:** React, Vite
- **Database:** MySQL (via XAMPP)
